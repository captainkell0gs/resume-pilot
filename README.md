# ResumePilot — React CV Builder

A multi-step resume builder built with React.
Users can enter personal details, add experiences, preview different templates, and export a professional CV as a PDF.

## Features

- Multi-step form
- Dynamic experience sections
- Multiple CV templates
- Live preview
- Auto scaling A4 layout
- PDF export
- Responsive UI
- Clickable progress navigation

## Tech Stack

React
CSS
html2pdf.js
Lucide Icons
